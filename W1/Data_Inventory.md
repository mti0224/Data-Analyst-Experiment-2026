# Data Inventory

## 專案資料來源
本專案使用 YouTube Studio 匯出的真實頻道資料作為分析來源，不使用教學案例資料。

## 已規劃匯出的資料類型

### 1. 影片成效報表
可能包含欄位：
- 影片標題
- 發布日期
- 觀看次數
- 曝光次數
- 曝光點閱率
- 平均觀看時長
- 觀看時間（小時）
- 按讚數
- 留言數
- 分享次數
- 訂閱人數變化

用途：
- 分析影片表現
- 建立基礎 SQL 查詢

### 2. 觀眾互動數據
可能包含欄位：
- 發布日
- 影片時長
- 按讚數
- 觀看時數
- 曝光次數
- 曝光率

用途：
- 分析觀眾黏著度
- 驗證內容互動假設
- 找出高表現影片特徵

### 3. 流量來源
可能包含欄位：
- 流量來源類型
- 外部來源
- YouTube 搜尋
- 建議影片
- 瀏覽功能
- 直接/未知來源
- 各來源觀看次數
- 各來源觀看時間
- 各來源平均觀看時長

用途：
- 分析頻道流量結構
- 比較自然流量與外部導流效果
- 支援內容分發優化建議

## 時間範圍
流量期間：
- 起始：2025/7/1
- 結束：2025/12/31

## 隱私處理計畫

### 可公開
- 影片標題
- 發布日期
- 觀看次數
- 按讚數
- 留言數
- 平均觀看時長
- 流量來源分類（若不涉及敏感收益資訊）
- 匿名化後的聚合統計結果
- SQL / Python 分析程式碼

### 不公開
- 收益相關欄位
